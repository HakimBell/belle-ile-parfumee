Contexte

Mon activité

Auto-entrepreneur depuis 10 ans dans la vente de parfums de grandes marques (Chanel, Dior, YSL, Guerlain...)

Le processus actuel

  • Réapprovisionnement chaque dimanche chez mon grossiste
  • Stock variable : dépend de ce que le grossiste reçoit du port du Havre le vendredi
  • Pas de visibilité à l'avance sur les références disponibles
  • Les clients me contactent le dimanche soir pour connaître les nouveautés

Problématique

Surcharge de travail

  • Inventaire manuel chaque dimanche
  • Mise à jour des produits un par un
  • Emails individuels à chaque client
  • Fichiers Excel à maintenir manuellement

Problématique

Conséquences

  • Processus chronophage
  • Risque d'erreurs humaines
  • Mauvaise expérience client
  • Perte de temps sur les tâches répétitives

Solution : Belle Île Parfumée

Pour les clients

  • Consulter le catalogue en temps réel
  • Voir les stocks disponibles
  • Passer commande en ligne 24/7
  • Plus besoin d'attendre mes emails

Pour l'administrateur

  • Mise à jour des produits centralisée
  • Gestion des commandes simplifiée
  • Fin des emails individuels
  • Gain de temps considérable

Objectifs

Améliorer l'efficacité

Automatiser la gestion des stocks et la communication des nouveaux produits disponibles

Centraliser les commandes

Mieux répondre aux attentes des clients tout en réduisant les tâches manuelles répétitives

Offrir une meilleure expérience utilisateur

Les clients peuvent consulter directement les produits disponibles et passer commande à leur convenance

Ajouter au Panier

Code Backend

CartService.java

public CartResponseDTO addItemToCart(String email, String productCode, int quantity) {
    // 1. Récupérer ou créer le panier (status = PENDING)
    Order cart = getOrCreateCart(email);

    // 2. Vérifier le stock disponible
    Product product = productRepository.findById(productCode)
        .orElseThrow(() -> new RuntimeException("Produit non trouvé"));

    if (product.getStock() < quantity) {
        throw new RuntimeException("Stock insuffisant");
    }

    // 3. Ajouter ou mettre à jour la ligne de commande
    OrderLine line = findOrCreateOrderLine(cart, product);
    line.setQuantity(line.getQuantity() + quantity);
    line.setUnitPrice(product.getPrice());

    orderLineRepository.save(line);
    return toCartResponseDTO(cart);
}

Code Frontend

CartContext.tsx

const addToCart = async (product: Product, quantity: number) => {
    if (isAuthenticated) {
        // Mode connecté : appel API REST
        const email = authService.getUserEmail();
        const response = await cartService.addItem(email, product.productCode, quantity);
        setItems(response.items);
    } else {
        // Mode non connecté : localStorage
        const existingItem = items.find(i => i.product.productCode === product.productCode);
        if (existingItem) {
            existingItem.quantity += quantity;
        } else {
            items.push({ product, quantity });
        }
        localStorage.setItem('cart', JSON.stringify(items));
        setItems([...items]);
    }
};

Sécurité

Authentification

  • JWT (JSON Web Token)
  • Cookie HttpOnly
  • Expiration : 24 heures
  • BCrypt (hash mots de passe)
  • Refresh token possible

Autorisation

  • Rôles : CLIENT, ADMIN
  • Routes protégées par rôle
  • Filtre JWT personnalisé
  • CORS configuré
  • Validation des entrées

Génération du Code Produit

Format : XXX-TTT-MMM-YYYY

Segment Description Exemple
XXX Code marque (3 lettres) CHA, DIO, YSL
TTT Type de concentration EDP, EDT, EXT
MMM Taille en ml 050, 100, 075
YYYY Numéro séquentiel 0001, 0002

Exemple : CHA-EDP-100-0001 → Chanel N°5 EDP 100ml
